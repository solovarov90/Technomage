# 🧿 ВВОДНЫЙ КОНЦЕПТ #002: Код Рождения: Твой Серийный Номер 🧬🔢🛡️

Каждый объект в Матрице имеет свой уникальный **ID**. Твой ID — это твоя дата рождения. В системе **Eidos.is** мы называем это **Серийным Номером Аватара**.

### Твоя Идентификация (Пример 09.07.1990):
- **ID 9 (User Interface):** Это твоя маска. Как ты контактируешь с реальностью. 
- **ID 1 (Kernel/Core):** Это твоя базовая мощность процессора. Твой «железный» потенциал.
- **ID 8 (Runtime Mission):** Твоя главная задача. Алгоритм, за выполнение которого тебе отсыпают ресурсы.

**🧿 Запомни: Система не знает твоего имени. Она знает только твой Код.** Если твой Код выдает ошибку — твой аккаунт в Матрице уходит в бан (неудачи). Мы здесь для того, чтобы этот Код поправить. 📡 [[reply_to_current]]
