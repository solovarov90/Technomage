# Поток: Режим Сверхпроводимости

### Техническая суть (Superconductivity)
Поток — это состояние, когда сопротивление среды и сознания падает до нуля.
* **Ping:** 0 ms. Задержки между мыслью и действием нет.
* **CPU Load:** 100% эффективность, но без перегрева.
* **Ego:** Off. "Тебя" нет. Есть только процесс (код пишется сам, танец танцуется сам).

### Условия входа (Login)
Чтобы войти в Поток, нужно сбалансировать два параметра:
1.  **Сложность задачи:** Должна быть чуть выше твоих навыков (вызов).
2.  **Навык:** Должен быть достаточным, чтобы принять вызов.
* *Если задача слишком легкая -> Скука.*
* *Если слишком сложная -> Тревога ([[Сопротивление]]).*
* *Баланс -> Поток.*

### Духовный аспект (У-Вэй)
В даосизме это называется **У-Вэй** (Деяние через Недеяние).
Ты не "делаешь" работу, ты позволяешь силе ([[Бог]] / [[Эволюция]]) действовать *через* тебя.
Ты становишься идеальным проводником.

> "В Потоке время исчезает. Ты не работаешь, ты играешь джаз с Вселенной."
