### Hard Code vs Soft Code
Твоя жизнь — это не прямая линия, а дерево вариантов (Decision Tree).
1.  **Hard Code ([[Судьба]] / [[Карма]]):** Константы, которые нельзя изменить. Родители, ДНК, страна рождения. Это дизайн уровня, в который тебя поместили.
2.  **Soft Code ([[Выбор]]):** Переменные. Твои реакции, решения и отношение. Это то, как ты проходишь этот уровень.

### Ловушка NPC
90% людей живут по "Дефолтным сценариям", загруженным в [[Подсознание]] в детстве.
* *Скрипт:* "Учись -> Работа -> Кредит -> Пенсия".
* Пока ты не осознан, ты просто выполняешь этот алгоритм. Ты — NPC.

### Как взломать Сценарий? (Переход на новую ветку)
Ты не можешь изменить сценарий, действуя логично. Логика — это часть старого кода.
Чтобы перейти на новую [[Временные линии]], нужно совершить **Сбой в Матрице**:
1.  Сделать то, что тебе не свойственно (выйти из паттерна).
2.  Перестать реагировать привычными эмоциями на проблемы (не кормить [[Маятники]]).

> "Сценарий — это не приговор. Это настройки по умолчанию.
