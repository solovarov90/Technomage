# Хроники Акаши как База Данных

**[[Хроники Акаши]]** — это вселенская база данных, где записано всё.
Научный аналог:
* **[[Квантовая теория информации]]**: информация — строительный блок реальности.
* **[[Голографическая Вселенная]]**: наш 3D мир — это проекция данных с 2D поверхности.

Вся инфа о нас хранится "в облаке". Древние называли это Акаши, мы называем Голограммой.
